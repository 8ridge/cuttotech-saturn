# 🎨 Система управления контентом (CMS)

## Что это?

Теперь ваш URL Shortener поддерживает **динамическое управление текстами** через Supabase! 

Вы можете **редактировать любые тексты на сайте** через админ-панель без необходимости:
- ❌ Редактировать код
- ❌ Пересобирать приложение  
- ❌ Перезагружать сервер

Просто откройте админ-панель, отредактируйте текст и **изменения применятся мгновенно** для всех пользователей! 🚀

## 📋 Быстрый старт

### 1. Создайте таблицу в Supabase

Откройте **Supabase Dashboard** → **SQL Editor** и выполните скрипт из `CONTENT_QUICK_START.md`

### 2. Настройте email администратора

В файле `/components/Header.tsx` замените:
```typescript
const isAdmin = user?.email === 'admin@yourdomain.com';
```
На свой email.

### 3. Войдите как админ

После входа в шапке появится красная кнопка **"Админ"** → откройте **"Управление контентом"**

## ✨ Возможности

- 🔍 **Поиск** по ключу, значению или описанию
- 📁 **Фильтрация** по категориям (home, auth, dashboard, admin, header)
- ✏️ **Inline редактирование** - редактируйте прямо в интерфейсе
- ⚡ **Мгновенное применение** - изменения видны сразу
- 🔄 **Автоматическое обновление** - пользователи видят новые тексты при следующей загрузке
- 💾 **Fallback режим** - если таблица не настроена, используются тексты из кода

## 📚 Документация

- `CONTENT_QUICK_START.md` - быстрая настройка за 3 шага
- `CONTENT_SETUP.md` - детальная настройка с полным SQL
- `CONTENT_MANAGEMENT_GUIDE.md` - полное руководство пользователя

## 🎯 Примеры использования

### Изменить название сервиса
1. Найдите `header.logo` в админ-панели
2. Измените "ShortURL" на своё название
3. Сохраните - готово!

### Изменить главный заголовок
1. Найдите `home.hero.title`  
2. Напишите свой текст
3. Сохраните - изменения на главной странице!

### Добавить новый текст
1. Создайте запись в таблице `content`
2. Используйте в коде: `const text = useText('my.new.key')`

## 🔒 Безопасность

- ✅ Только администраторы могут редактировать
- ✅ Все могут читать (для отображения на сайте)
- ✅ История изменений через `updated_at`

## 🤝 Поддержка

Если возникли проблемы, проверьте:
1. Таблица `content` создана в Supabase
2. Email администратора правильно указан
3. RLS политики настроены
4. Консоль браузера на наличие ошибок
