# 🎯 Начало работы - Ubuntu 22 Server

Добро пожаловать! Это самый быстрый способ запустить URL Shortener на вашем Ubuntu 22 сервере.

## 🚀 Три простых шага

### 1️⃣ Скачайте проект

**Вариант A - Git (рекомендуется):**
```bash
git clone <URL_вашего_репозитория> url-shortener
cd url-shortener
```

**Вариант B - ZIP файл:**
```bash
# Скачайте ZIP на компьютер через GitHub кнопку "Code"
# Загрузите на сервер и распакуйте:
unzip url-shortener.zip
cd url-shortener
```

### 2️⃣ Запустите установку

```bash
chmod +x install.sh
./install.sh
```

Скрипт автоматически установит всё необходимое!

### 3️⃣ Запустите сайт

```bash
npm run preview -- --host 0.0.0.0 --port 3000
```

**Готово!** Сайт доступен: `http://ваш-ip:3000`

## 🔧 Управление сервисом

После настройки автозапуска (см. UBUNTU_INSTALL.md):

```bash
# Запуск
sudo systemctl start url-shortener

# Остановка
sudo systemctl stop url-shortener

# Перезапуск
sudo systemctl restart url-shortener

# Статус
sudo systemctl status url-shortener

# Логи
sudo journalctl -u url-shortener -f
```

## ❓ Часто задаваемые вопросы

### Как изменить порт?
```bash
npm run preview -- --host 0.0.0.0 --port 4000
```

### Как настроить свой Supabase?
Отредактируйте файл `utils/supabase/info.tsx`

### Сайт недоступен извне?
```bash
sudo ufw allow 3000/tcp
```

### Как обновить проект?
```bash
git pull
npm install
npm run build
sudo systemctl restart url-shortener
```
