## 🎬 Пошаговый план

### Шаг 1: Установите зависимости
```bash
npm install
```

### Шаг 2: Настройте Supabase
- Создайте проект на [supabase.com](https://supabase.com)
- Скопируйте Project ID и anon key
- См. подробности в [QUICK_START.md](./QUICK_START.md)

### Шаг 3: Обновите конфигурацию
- Откройте `/utils/supabase/info.tsx`
- Замените placeholder значения на ваши

### Шаг 4: Настройте БД
- Выполните SQL скрипт в Supabase Dashboard
- Скрипт есть в [QUICK_START.md](./QUICK_START.md)

### Шаг 5: Запустите!
```bash
npm run dev
```

**Готово!** Откройте http://localhost:3000 🎉

## 🆘 Нужна помощь?

### Частые вопросы

**Q: Не устанавливаются зависимости**
```bash
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

**Q: Не видно текстов на сайте**
- Проверьте что SQL скрипт выполнен в Supabase
- Проверьте консоль браузера (F12)
- Убедитесь что credentials правильные

**Q: Где взять Supabase credentials?**
1. [supabase.com](https://supabase.com) → Войти
2. Выбрать проект или создать новый
3. Settings → API → скопировать данные
