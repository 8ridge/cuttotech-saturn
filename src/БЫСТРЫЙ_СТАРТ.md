# 🚀 Быстрый старт на Ubuntu 22

Простая инструкция для запуска проекта за 5 минут.

## Шаг 1: Скачайте проект на сервер

### Через Git:
```bash
git clone <URL_репозитория> url-shortener
cd url-shortener
```

### Через ZIP:
```bash
# Скачайте ZIP через кнопку "Code" на GitHub
# Загрузите на сервер и распакуйте:
unzip url-shortener.zip
cd url-shortener
```

## Шаг 2: Запустите автоматическую ��становку

```bash
chmod +x install.sh
./install.sh
```

Скрипт автоматически:
- Установит Node.js (если не установлен)
- Установит все зависимости
- Соберет проект

## Шаг 33: Запустите проект

```bash
npm run preview -- --host 0.0.0.0 --port 3000
```

Готово! Сайт доступен: `http://ваш-ip:3000`

## 🔄 Автозапуск (опционально)

Если хотите, чтобы сайт запускался автоматически:

1. Создайте service файл:
```bash
sudo nano /etc/systemd/system/url-shortener.service
```

2. Вставьте (замените `/путь/к/проекту` и `username`):
```ini
[Unit]
Description=URL Shortener
After=network.target

[Service]
Type=simple
User=username
WorkingDirectory=/путь/к/проекту
ExecStart=/usr/bin/npm run preview -- --host 0.0.0.0 --port 3000
Restart=on-failure

[Install]
WantedBy=multi-user.target
```

3. Активируйте:
```bash
sudo systemctl daemon-reload
sudo systemctl enable url-shortener
sudo systemctl start url-shortener
```

## 📋 Полезные команды

```bash
# Проверить статус
sudo systemctl status url-shortener

# Посмотреть логи
sudo journalctl -u url-shortener -f

# Перезапустить
sudo systemctl restart url-shortener

# Остановить
sudo systemctl stop url-shortener
```

## ❗ Решение проблем

### Порт 3000 занят?
```bash
# Используйте другой порт
npm run preview -- --host 0.0.0.0 --port 4000
```

### Сайт недоступен?
```bash
# Откройте порт в firewall
sudo ufw allow 3000/tcp
```

### Node.js не установлен?
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```
