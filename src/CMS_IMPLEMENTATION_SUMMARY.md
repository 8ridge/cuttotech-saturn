## 🗂️ Структура контента в Supabase

### Таблица `content`
```sql
- id (UUID) - первичный ключ
- key (TEXT) - уникальный идентификатор текста
- value (TEXT) - сам текст
- category (TEXT) - категория для группировки
- description (TEXT) - описание для админа
- created_at (TIMESTAMP) - время создания
- updated_at (TIMESTAMP) - время обновления (автоматически)
```

### Категории контента
- **home** - главная страница (106 текстов)
- **header** - шапка сайта (3 текста)
- **auth** - авторизация (18 текстов)
- **dashboard** - личный кабинет (12 текстов)
- **admin** - админ панель (6 текстов)

### Примеры ключей
- `home.hero.title` - главный заголовок
- `home.hero.subtitle` - подзаголовок
- `header.logo` - название сервиса
- `auth.benefits.links` - преимущество - ссылки
- `dashboard.stats.links` - статистика ссылок

## 🔐 Безопасность

### Row Level Security (RLS)
- ✅ Все пользователи могут **читать** контент
- ✅ Только администраторы могут **редактировать**
- ✅ Проверка через `auth.jwt() ->> 'email'`

### Настройка администратора
Замените в двух местах:

1. **Supabase RLS политика:**
```sql
auth.jwt() ->> 'email' = 'admin@yourdomain.com'
```

2. **Header.tsx:**
```typescript
const isAdmin = user?.email === 'admin@yourdomain.com';
```

## 🚀 Как использовать

### Доступ к админ-панели
1. Войдите на сайт под учетной записью администратора
2. В хедере появится кнопка **"Админ"** (красная)
3. Откройте админ-панель
4. Перейдите на вкладку **"Управление контентом"**

### Редактирование текста
1. Найдите нужный текст (поиск или фильтры)
2. Нажмите **"Редактировать"**
3. Измените текст
4. Нажмите **"Сохранить"**
5. ✨ Изменения применяются мгновенно!

### Добавление нового текста в код

```tsx
import { useText } from '../utils/content';

function MyComponent() {
  const myTitle = useText('my.section.title', 'Fallback заголовок');
  
  return <h1>{myTitle}</h1>;
}
```

### Добавление нового текста в БД

**Через SQL:**
```sql
INSERT INTO content (key, value, category, description) 
VALUES (
  'my.section.title',
  'Мой заголовок',
  'home',
  'Заголовок моей новой секции'
);
```

**Через Supabase Dashboard:**
Table Editor → content → Insert row

## 🐛 Troubleshooting

### Таблица не найдена
```
Ошибка: relation "content" does not exist
```
**Решение:** Выполните SQL из `CONTENT_QUICK_START.md`

### Нет кнопки "Админ"
**Проверьте:**
- Email совпадает в Header.tsx
- Вы вошли под правильной учетной записью
- Консоль браузера на ошибки

### Изменения не сохраняются
**Проверьте:**
- RLS политики настроены правильно
- Email администратора совпадает в политике
- У вас есть права администратора

### Контент не загружается
**Проверьте:**
- Supabase credentials правильные
- Таблица создана и доступна
- Консоль браузера на ошибки
- Если таблица не настроена - используются fallback тексты
